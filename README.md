# CV — Furkan Doğmuş

DevOps Engineer — Overleaf/LaTeX Oxford Formatı

## Kullanım

1. [Overleaf](https://www.overleaf.com/)'e git
2. **New Project** → **Upload Project**
3. `cv-furkan-dogmus.tex` dosyasını yükle
4. Compiler: **XeLaTeX** veya **pdfLaTeX** seç
5. **Recompile** — PDF çıktısı hazır!

## Gerekenler (Overleaf'te otomatik)

- `moderncv` paketi (moderncv templates)
- `fontenc`, `geometry` (standart)
